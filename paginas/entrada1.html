WP Rocket, optimizar WordPress al alcance de todos

WP Rocket es un plugin que permite implementar caché de página, minificar archivos CSS y Javascript, combinar archivos CSS y JS, aplicar algunos resource hints y optimizar el código servido, entre otras cosas. Pero carece de caché de objetos en Memcached o Redis, así como de caché de consultas en Memcached o Redis, etc. Por esta razón, no creo que WP Rocket sea el mejor plugin de caché para WordPress, pero sí puedo decir que es el que se suele adaptar mejor en la mayoría de webs simples: blogs, webs corporativas, ecommerce pequeños, etc. ¿Por qué WP Rocket y no otro? Pues porque no he encontrado nada igual a la hora de optimizar una página WordPress.

¿Qué hace WP Rocket?

WP Rocket es lo que llamamos en WordPress, de forma general, un plugin de caché. Si somos un poco más específicos, podemos decir que es un plugin de caché de página guardado en disco con funcionalidades de optimización de archivos CSS y JS.

Adicionalmente, permite implementar unas cuantas técnicas WPO además del caché:
  2. Optimizar el CSS y JS: minificar, combinar, carga asíncrona, etc.
  3. Carga retrasada de los scripts Javascript.
  4. Limpieza de las clases CSS no utilizadas en la carga de la web.
  5. Implementación de Lazy Load en imágenes e iframes.
  6. Mejora del CLS añadiendo las dimensiones como atributo a las imágenes.
  7. Implementación de un CDN: propio o RocketCDN.
  8. Gestión de los pulsos del API Heartbeat o admin-ajax.php-
  9. Limpieza de la base de datos.
Configurar cache de página en WordPress Precisamente el caché de página es el punto fuerte de WP Rocket, es la razón por la que nació, y la verdad es que lo hace muy bien. Alguna prueba de rendimiento y respuesta he hecho a lo largo de estos años: en todas he visto que WP Rocket sirve el caché de página de forma muy optimizada.
